How PixelSqueeze Works?

What is a Digital Image? +

A digital image is a grid of pixels, each a colored dot defined by RGB values (0-255). A 1920x1080 image has over 2 million pixels, creating large data sizes.

Why Compress Images? +

Compression reduces file size by removing redundant data, ideal for web or mobile use while preserving quality.

How Does SVD Work? +

SVD decomposes a matrix into:

  • U: Spatial patterns.
  • S: Importance values.
  • V: Feature patterns.

Keeping top k values approximates the matrix, balancing quality and size.

PixelSqueeze Process +

Applies SVD to RGB channels:

  1. Decomposes into U, S, V.
  2. Retains top k values.
  3. Reconstructs with metrics.
