The Science Behind the Squeeze

From pixels to matrices, explore the linear algebra that powers smart image compression.

A digital image is fundamentally a grid of pixels. For a color image, this grid is actually three-dimensional. It can be represented as a large matrix where each entry corresponds to a pixel's color. This color is typically defined by three intensity values for Red, Green, and Blue (RGB), each ranging from 0 to 255. Therefore, a color image of size $H \times W$ (Height x Width) can be seen as three separate $H \times W$ matrices, one for each color channel.

Singular Value Decomposition (SVD) is a powerful matrix factorization technique from linear algebra. It states that any matrix $A$ can be broken down into three other matrices: $$ A = U \Sigma V^T $$

  • $U$: An orthogonal matrix whose columns (left singular vectors) describe the spatial patterns in the image.
  • $\Sigma$: A diagonal matrix containing the singular values, sorted by importance.
  • $V^T$: The transpose of an orthogonal matrix whose rows (right singular vectors) describe feature patterns.

The magic of SVD for compression lies in the $\Sigma$ matrix. By keeping only the top $k$ singular values and their corresponding vectors, we create a very good, low-data approximation of the original image: $$ A \approx U_k \Sigma_k V_k^T $$ This requires storing far less data than the original matrix $A$, thus achieving compression.

  1. Decomposition: The uploaded image is separated into its Red, Green, and Blue channel matrices. SVD is performed on each.
  2. Reduction (Approximation): Based on your selected quality ($k$), we keep only the top $k$ singular values and vectors for each channel.
  3. Reconstruction: The smaller matrices are multiplied to reconstruct the compressed channels, which are then combined.
  4. Evaluation: We calculate the Compression Ratio and PSNR to show you the results of the compression.
